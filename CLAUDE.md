# Claude Code Rules

## solutions/ フォルダについて

`solutions/` 配下のファイルはユーザー自身が書いたコードの記録です。

**Claude Code はこのフォルダ内のファイルを絶対に編集・修正してはならない。**

- コードに誤りがあっても指摘にとどめ、ファイルを書き換えない
- ユーザーが明示的に「修正して」と指示した場合のみ編集可能

## フォルダ構成

```
solutions/
  data_structures/   # データ構造の問題
  algorithms/        # アルゴリズムの問題（予定）
notes/               # メモや解説
```
