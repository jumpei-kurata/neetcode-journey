## NeetCode 150

## 使い方

- チェックボックス `- [ ]` をGitHub上でクリックするか、Cursorで `- [ ]` を `- [x]` に書き換える
- 問題を解いたら末尾にメモを追加: 解いた日 / 時間計算量 / 空間計算量 / 一言コメント
- 復習が必要な問題は末尾に `🔁復習` のような印をつける（絵文字は任意）
- 各問題のリンクをクリックするとLeetCodeに飛ぶ
- `(Premium)` と付いた問題はLeetCode有料プラン限定。NeetCode.ioの該当ページから無料で見れる代替サイトへのリンクがあることが多い

メモの書き方（例）:

```
- [x] [Two Sum](https://leetcode.com/problems/two-sum/) — Easy — 解: 4/23 / O(n) / O(n) / ハッシュで一発
- [x] [Valid Anagram](https://leetcode.com/problems/valid-anagram/) — Easy — 解: 4/24 / O(n) / O(1) / 26文字カウント
```

## 学習ルール（自分ルール）

1. 新規問題は**15〜30分自力で考える**。それでも無理なら解説動画
2. 解説を見た場合は、**何も見ずに再実装**するまで完了にしない
3. 解けたら Cursor に貼って `solutions/<category>/<problem>.py` として保存
4. Tracker を更新 → git commit。コミットメッセージは `solved: <Problem Name>` で統一
5. 解いた3日後、1週間後、1ヶ月後に復習
6. 復習で解けなかった問題は「復習キュー」に戻す
